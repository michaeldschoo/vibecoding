Chint DTSU666은 태양광 시스템에서 전력 흐름을 정밀하게 측정하는 3상 스마트 미터입니다. 이 가이드는 미터기의 LCD 화면을 통해 주요 전력 데이터를 확인하는 방법을 보여줍니다.

버튼 조작 방법

3화살표 버튼 (→): 다음 항목으로 이동 또는 값 증가

미터기가 거꾸로 설치된 경우

현재 미터기가 물리적으로 거꾸로 설치되어 있다면, 코드 30번(총 유효 전력)의 부호(+/-)가 실제와 반대로 표시됩니다:

코드 30번 해석 방법

플러스(+) 표시: 실제로는 태양광이 생산되어 그리드로 수출 중
마이너스(-) 표시: 실제로는 집에서 전기를 사용하여 그리드에서 수입 중

설치 오류 진단 가이드

1단계: 낮 시간 (태양광 생산 중)

코드 30번을 확인하세요. 플러스(+) 값이 높게 나온다면, 미터기가 거꾸로 설치되었을 가능성이 높습니다.

2단계: 밤 시간 (전기 사용 중)

EV 충전(1.3kW) 같은 큰 부하를 켜고 코드 30번을 확인하세요. 마이너스(-) 값이 나온다면 설치 오류가 확실합니다.

3단계: 화살표 방향 확인

LCD 화면의 화살표 방향과 코드 30번의 부호가 상식적으로 맞지 않는다면, 설치 업체에 연락하여 미터기 재설치를 요청하세요.

문제 해결

미터기가 거꾸로 설치되어 있다면, 설치 업체에 다음과 같이 요청하세요:

"DTSU666 미터기(N1.2)가 물리적으로 거꾸로 설치되어 있습니다. 올바른 방향으로 재설치해 주십시오."
